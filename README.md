# Tulane Sports Medicine (AT)

Athletic training scheduling — SwiftUI iOS app built for Tulane Sports Medicine.

## Open in Xcode

```
iOS/TulaneSportsMedicine!/TulaneSportsMedicine!.xcodeproj
```

Requires **Xcode 15+** and **iOS 17+**.
